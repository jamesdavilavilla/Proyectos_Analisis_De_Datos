# ⏱️ Análisis de Series Temporales - Template

## Componentes de Series Temporales
- **Tendencia:** Dirección general a largo plazo
- **Estacionalidad:** Patrones recurrentes regulares
- **Ciclos:** Fluctuaciones irregulares de largo plazo
- **Ruido:** Variación aleatoria

## Modelos de Forecasting
- **ARIMA:** AutoRegressive Integrated Moving Average
- **Exponential Smoothing:** Holt-Winters
- **Prophet:** Facebook's time series tool
- **LSTM:** Deep learning for sequences

## Validación de Forecasts
- **Mean Absolute Error (MAE)**
- **Mean Squared Error (MSE)**
- **Mean Absolute Percentage Error (MAPE)**
- **Out-of-sample testing**
- **Cross-validation temporal**

## Detección de Anomalías
- Statistical methods (Z-score, IQR)
- Isolation Forest
- Seasonal decomposition
- Prophet anomaly detection
