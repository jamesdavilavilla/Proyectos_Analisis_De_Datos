# ✅ Checklist de Presentación - Template

## Pre-Presentación

### Preparación de Contenido
- [ ] **Audiencia identificada:** Nivel, intereses, tiempo disponible
- [ ] **Objetivo claro:** ¿Qué decisión/acción queremos generar?
- [ ] **Story arc definido:** Contexto → Conflicto → Resolución
- [ ] **Insights prioritizados:** Los 3 más importantes
- [ ] **Datos validados:** Fuentes confiables, cálculos verificados

### Preparación Visual
- [ ] **Charts optimizados:** Tipo correcto para el mensaje
- [ ] **Colors consistentes:** Brand guidelines seguidos
- [ ] **Text legible:** Tamaño apropiado para el venue
- [ ] **Annotations claras:** Key insights destacados
- [ ] **Flow lógico:** Progresión natural de ideas

### Preparación Técnica
- [ ] **Backup creado:** Múltiples formatos disponibles
- [ ] **Connectivity tested:** Internet, proyector, audio
- [ ] **Timing rehearsed:** Dentro del tiempo asignado
- [ ] **Q&A prepared:** Anticipar preguntas difíciles

## Durante la Presentación

### Opening (2-3 minutos)
- [ ] **Hook strong:** Captar atención inmediatamente
- [ ] **Agenda clear:** Qué cubriremos
- [ ] **Time frame:** Duración y tiempo para preguntas

### Main Content (80% del tiempo)
- [ ] **Logical flow:** Un punto lleva al siguiente
- [ ] **Data storytelling:** No solo mostrar, sino explicar
- [ ] **Audience engagement:** Preguntas retóricas, pauses
- [ ] **Time management:** Monitorear progreso

### Closing (2-3 minutos)
- [ ] **Key takeaways:** Resumir los 3 insights principales
- [ ] **Call to action:** Qué hacer next
- [ ] **Contact info:** Cómo seguir la conversación

## Post-Presentación

### Immediate Follow-up
- [ ] **Notes captured:** Feedback y preguntas recibidas
- [ ] **Action items:** Quién hace qué cuándo
- [ ] **Materials shared:** Slides, data, additional resources
- [ ] **Thank you sent:** A organizadores y participantes

### Long-term Follow-up
- [ ] **Impact measured:** ¿Se tomaron las decisiones esperadas?
- [ ] **Lessons learned:** Qué mejorar para próxima vez
- [ ] **Relationships maintained:** Seguimiento con stakeholders
- [ ] **Content updated:** Incorporar feedback para futuras presentaciones
